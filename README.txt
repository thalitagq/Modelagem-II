Teclas:
a - Após a determinação dos pontos, pressionando a tecla a, o vetor inicial é mostrado na tela.
s - Pressionando a tecla s, o fecho é calculado.
f - Abre modal para escolher um arquivos com pontos para calcular o fecho
    Os pontos devem estar como se segue :
        100 50    //primeiro ponto
        30  20    // segundo ponto 
        ...  
    sendo a primeira string da esquerda para direita a coordenada x do ponto e a segunda string da mesma linha é y,    
x - Fecha modal
d - Faz download de um arquivo .obj
